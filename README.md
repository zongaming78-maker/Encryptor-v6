## 🛡️ How Encryption Works

Encryption changes your readable text into a scrambled format. Think of it as a digital lock. You provide a password, and the program uses this password to mix your text. Without the correct password, the scrambled blob looks like random characters. This process stops unauthorized people from seeing your private notes.

This program uses AES-256-GCM technology. This is a standard for security in the computer industry. It ensures that your data stays private and that nobody changes your original text without your knowledge.
